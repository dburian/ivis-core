    public setView(xMin: *, xMax: *, yMin: *, yMax: *, source: *, causedByUser: *) source

    Set the visible region of the chart to defined limits (in units of the data, not in pixels). If the axis data type is keyword (string), both boundary values are included.

    Params:

    NameTypeAttributeDescription
    xMin *

    left boundary of the visible region (in units of data on x-axis)
    xMax *

    right boundary of the visible region (in units of data on x-axis)
    yMin *

    bottom boundary of the visible region (in units of data on x-axis)
    yMax *

    top boundary of the visible region (in units of data on x-axis)
    source *

    the element which caused the view change (if source === this, the update is ignored)
    causedByUser *

    tells whether the view update was caused by user (this propagates to props.viewChangeCallback call), default = false
