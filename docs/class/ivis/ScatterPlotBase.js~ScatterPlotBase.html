    public setView(xMin: *, xMax: *, yMin: *, yMax: *, source: *, causedByUser: *, withTransition: *) source

    Set the visible region of the chart to defined limits (in units of the data, not in pixels)

    Params:

    NameTypeAttributeDescription
    xMin *

    left boundary of the visible region (in units of data on x-axis)
    xMax *

    right boundary of the visible region (in units of data on x-axis)
    yMin *

    bottom boundary of the visible region (in units of data on y-axis)
    yMax *

    top boundary of the visible region (in units of data on y-axis)
    source *

    the element which caused the view change (if source === this, the update is ignored)
    causedByUser *

    tells whether the view update was caused by user (this propagates to props.viewChangeCallback call), default = false
    withTransition *

    set to true if view change should be animated (props.withTransition must be also true), default = false
