    MinMaxLoader

    Extends:

    react~Component → MinMaxLoader

    This component fetches the minimum and maximum for given signal(s) and calls props.processData with the values. The values are given in form: { signalId: { min, max } }. Time interval filtering (using TimeContext) is also supported. It has no UI.

    Decorators:

    • withComponentMixins([withTranslation, withErrorHandling, intervalAccessMixin()], ["reloadData"])
